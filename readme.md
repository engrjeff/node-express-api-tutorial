# Node Express Backend

## Clone the Repo

```shell
git clone https://github.com/engrjeff/node-express-api-tutorial.git
```

## Install dependencies

```shell
npm install
```

## Create `.env` file

```shell
// .env

PORT=5010
```

## Run Dev Server

```shell
npm run dev
```

## Run Linter

```shell
npm run lint
```
